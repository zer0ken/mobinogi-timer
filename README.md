# 모비노기 각성 타이머 — 수동 버전

마비노기 모바일 각성 패시브의 지속시간과 쿨다운을 추적하는 오버레이 타이머입니다.

## 기능

- **단축키 시작** — 설정한 단축키(기본값: ` 키)를 눌러 타이머 시작
- **단축키 변경** — 원하는 키로 변경 가능
- **엠블럼 선택** — 8종 엠블럼 중 사용하는 엠블럼을 선택
- **자동 반복** — 쿨다운 종료 시 자동으로 타이머 재시작
- **눈먼예언자 룬 지원** — 착용/초월 단계에 따라 쿨다운 자동 계산
- **오버레이 타이머 바** — 게임 화면 위에 항상 표시
- **투명도/너비 조절** — 타이머 바의 투명도와 너비 조절
- **위치 저장** — 타이머 바를 드래그하여 이동하면 위치 자동 저장

## 설치

[최신 릴리즈](https://github.com/zer0ken/mobinogi-timer/releases)에서 `mobinogi-timer-manual.exe`를 다운로드하여 실행하세요.

## 사용법

### 설정

1. **엠블럼** — 사용하는 엠블럼을 선택합니다.
2. **눈먼예언자** — 룬 착용 여부와 초월 단계를 선택합니다.
3. **단축키** — 타이머를 시작할 단축키를 설정합니다.
4. **자동 반복** — 쿨다운 종료 시 자동으로 타이머를 재시작합니다.
5. **투명도** — 타이머 바의 투명도를 조절합니다.
6. **바 너비** — 타이머 바의 너비를 조절합니다.

설정 창이 선택된 상태에서 타이머 바를 드래그하여 원하는 위치로 이동할 수 있습니다.

### 타이머

게임 플레이 중 각성을 사용할 때 설정한 단축키를 누르면 타이머가 시작됩니다.

| 색상 | 상태 |
|------|------|
| 파랑 | 각성 준비됨 |
| 하늘색 | 각성 효과 지속 중 |
| 빨강 | 각성 효과 10초 이하 |
| 회색 | 쿨다운 진행 중 |

## 직접 빌드하기

### 필요한 프로그램

- [Node.js](https://nodejs.org/) (v18 이상)
- [Rust](https://www.rust-lang.org/tools/install) (1.70 이상)
- Windows 10/11

### 빌드 방법

```bash
git clone https://github.com/zer0ken/mobinogi-timer.git
cd mobinogi-timer
git checkout manual
npm install
npx tauri build
```

완성된 파일 위치: `src-tauri/target/release/mobinogi-timer.exe`

### 개발 모드

```bash
npx tauri dev
```
