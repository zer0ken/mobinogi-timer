# 모비노기 각성 타이머

마비노기 모바일 각성 패시브의 지속시간과 쿨다운을 추적하는 오버레이 타이머입니다.

## 버전

이 프로젝트는 두 가지 버전으로 제공됩니다:

### 🎮 수동 버전 (manual)
- 설정한 단축키로 타이머 수동 시작
- 엠블럼을 직접 선택
- Npcap 설치 불필요
- **다운로드**: [최신 릴리즈](https://github.com/zer0ken/mobinogi-timer/releases) (`mobinogi-timer-manual.exe`)

### 🔍 자동 버전 (auto)
- 패킷 캡처로 각성 발동 자동 감지
- 엠블럼 종류 자동 인식
- Npcap 설치 필요
- **다운로드**: [최신 릴리즈](https://github.com/zer0ken/mobinogi-timer/releases) (`mobinogi-timer-auto.exe`)

## 개발

각 버전은 독립된 브랜치에서 개발됩니다:

- `main` — 프로젝트 소개 및 워크플로우
- `manual` — 수동 버전 소스 코드
- `auto` — 자동 버전 소스 코드

자세한 개발 가이드는 각 브랜치의 README를 참조하세요.
