# 모비노기 각성 타이머

마비노기 모바일 각성 패시브의 지속시간과 쿨다운을 추적하는 오버레이 타이머입니다.

## 기능

- **오버레이 타이머 바** — 게임 오버레이에 항상 표시되어, 게임 중에도 타이머를 확인할 수 있습니다.
- **단축키 지원** — 단축키 하나로 타이머를 시작/재시작합니다.
- **단축키 변경** — 기본 `` ` `` 키 외에 원하는 키로 변경할 수 있습니다.
- **자동 반복** — 쿨다운이 끝나면 자동으로 다시 지속시간을 카운트합니다.
- **투명도 조절** — 타이머 바의 투명도를 10~100% 범위에서 조절할 수 있습니다.
- **위치 저장** — 타이머 바를 드래그하여 이동하면 위치가 자동 저장됩니다.

## 사용법

1. `mobinogi-timer.exe`를 실행합니다.
2. 설정 창에서 지속시간, 쿨다운, 단축키 등을 설정한 뒤 **저장**을 클릭합니다.
3. 설정 창이 선택된 상태에서 타이머 바를 드래그하여 원하는 위치로 이동합니다.
4. 게임 중 설정한 단축키를 누르면 타이머가 시작됩니다.

| 색상 | 상태 |
|------|------|
| 파랑 | 각성 준비됨 |
| 초록 | 각성 효과 지속 중 |
| 빨강 | 각성 쿨다운 진행 중 |

## 직접 빌드하기

### 필요한 프로그램

- [Node.js](https://nodejs.org/) (v18 이상)
- [Rust](https://www.rust-lang.org/tools/install) (1.70 이상)
- Windows 10/11

### 빌드 방법

```bash
npm install
npx tauri build
```

완성된 파일 위치: `src-tauri/target/release/mobinogi-timer.exe`
